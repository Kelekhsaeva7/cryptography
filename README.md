# 🎓 Криптография для школьников 5-7 классов

Интерактивный образовательный сайт для изучения основ криптографии.

## 🚀 Быстрый старт

Просто откройте [index.html](index.html) в браузере или разместите на GitHub Pages.

## 📚 Содержание курса

1. **Введение в криптографию** - что это и зачем нужно
2. **Шифр Цезаря** - исторический шифр сдвига
3. **Шифр Атабаш** - зеркальный шифр
4. **Шифр Виженера** - полиалфавитный шифр
5. **Метод Казиски** - как взломать Виженера
6. **Энигма и XOR** - машинное шифрование
7. **Финальный квест** - практическое задание

## 🎯 Цели проекта

- Познакомить школьников с основами криптографии
- Развить логическое и алгоритмическое мышление
- Показать практическое применение математики
- Заинтересовать профессиями в области информационной безопасности

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- GitHub Pages для хостинга
